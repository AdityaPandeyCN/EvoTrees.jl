__EvoTrees.jl__ supports four model families:
    - EvoTreeRegressor
        - Linear (minimize mean-squared error)
        - Logistic (minimize cross-entropy)
        - L1: minimize mean-absolute error
        - Quantile: minimize mean-absolute off the quantile
    - EvoTreeClassifier
    - EvoTreeCount
    - EvoTreeGaussian
