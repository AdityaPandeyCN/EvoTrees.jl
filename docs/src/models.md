EvoTrees.jl supports four model families:
- EvoTreeRegressor
    - Linear (minimize mean-squared error)
    - Logistic (minimize cross-entropy)
    - L1: minimize mean-absolute error
    - Quantile
- EvoTreeClassifier
- EvoTreeCount
- EvoTreeGaussian
